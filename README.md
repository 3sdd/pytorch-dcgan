

# dcgan動かしてみた

https://pytorch.org/tutorials/beginner/dcgan_faces_tutorial.html

## dependencies

- pytorch
- torchvision
- matploblib



## run


```
python main.py --dataroot="path/to/images"
```



`dataroot`はデータセットへのパス。
(e.g. `E:\Datasets\Anime Face Dataset`)。



```

|- Anime Face Dataset
       |-  images
             |- image1.png
             |- image2.png
             |- ...


```
